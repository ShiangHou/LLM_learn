# 字符串解码

## 初步思路

很显然用栈，

注意这里有三种情况，一种是数字，一种是[],一种是字母

我们需要讨论这三个情况

说白了，当遇到]的时候，直接开始我们的替换过程，

首先先到第一个[,然后把里面的字符提取出来，然后把[前面的数字提取出来，算好后直接再压入栈就行

本质上，我们遍历完字符串，把]都处理完就行了

那么问题是，怎么判断一个字符串是不是数字呢

```python 
class Solution:
    def decodeString(self, s: str) -> str:
        #思路是，除了]都全部压入
        #遇到]后，到栈里面取，取到第一个次遇到[和前面的数字，计算下来，然后再压回栈
        stack = []
        for k in s:
            if k.isdigit() or k.isalpha() or k == "[":
                stack.append(k)
            elif k == "]":
                temp_alpha = ""#准备乘的字母
                while stack[-1] != "[":
                    temp_alpha += stack.pop()

                #栈是先进先出的，记得把temp_alpha 反过来
                temp_alpha = temp_alpha[::-1]
                #遇到了[，弹出[
                stack.pop()
                #找前面的数字,注意这里可能有两位数的或者多位数的，需要到判断出下一个不为数字，或者栈是空了
                temp_num = ""
                while stack and stack[-1].isdigit():
                    temp_num += stack.pop()
                temp_num = int(temp_num[::-1])
                temp = ""
                for _ in range(temp_num):
                    temp += temp_alpha
                #最后再压入栈
                stack.append(temp)
        return "".join(stack)#最后输出合并的栈里面的所有内容



```
