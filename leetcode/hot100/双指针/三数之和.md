# 三数之和
力扣15

## 初步思路

题目就是说，要从nums里面找到三个下标不同的数字，让他们的和是0

这里要求返回的是这些数，不是下标

思路有一点印象，是先固定住一个，然后l和r从所有的后面开始遍历，小了就l+，大了就l-，直到找到目标

那么为了让这个小了+大了-的操作，则可以先排序在做

```python 
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        n = len(nums)
        nums.sort()
        ans = []
        for i in range(n):
            l,r = i+1,n-1
            temp_sum = nums[i]
            while r>l:
                if nums[i]+nums[l]+nums[r] == 0:#搜集答案,
                    ans.append([nums[i],nums[l],nums[r]])
                    break
                elif nums[i]+nums[l]+nums[r]>0:
                    r -= 1
                else:
                    l+=1
        return ans 
```

上面这个有一点问题，就是会出现重复的，比如0 0 0 0，那么会多次把0都放进去，

重点在于去重上，

第一个去重是第一层的i，如果i和i-1是一样的，那么直接跳过就行

此外，后面两个也是需要去重，即如果r++完后发现还是和r一样，那么也去重，l也是同理

```python 
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        n = len(nums)
        nums.sort()
        ans = []
        for i in range(n):
            l,r = i+1,n-1
            temp_sum = nums[i]
            if i >0 and nums[i-1] == nums[i]:
                continue
            while r>l:
                if nums[i]+nums[l]+nums[r] == 0:#搜集答案,
                    ans.append([nums[i],nums[l],nums[r]])
                    #做一下这里的去重,找到第一个不一样的
                    while r>l and nums[r] == nums[r-1]:
                        r-=1
                    while r>l and nums[l] == nums[l+1]:
                        l+=1
                    #这里是处理不重复的，正常处理
                    r -= 1
                    l +=1
                elif nums[i]+nums[l]+nums[r]>0:
                    r-=1
                else:
                    l+=1

        return ans 

```

注意别忘了不重复的，此外后面的l和r的重复只需要看找到答案的时候就行
