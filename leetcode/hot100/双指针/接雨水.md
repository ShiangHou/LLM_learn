# 接雨水
力扣42

## 栈解法
经典题目，首先我们定义一个东西，即下标i位置头顶能放多少水

我们看0到i-1位置的最大，假设是20，和i+1到n的位置上的最大，假设是30，那么在左边，我的i头顶的水可以高是20，因为有左边那个20的给我兜住，同理右边也是，那么我的i头顶的就是左右两边最大的取小的，即min（max_left,max_right）-nums[i]，但是注意一下，如果numsi比左右最大的还要大，那么i头顶其实是0哈哈哈，那么直接就和0比一下就行

max(  min（max_left,max_right）-nums[i] ,0    )

所以现在我们需要找左边0到i-1的最大和右边i+1到n的最大，我们可以用一个新的数据结构来存就行了

0位置直接导入，压入新的，如果新的没有顶的大，直接复用顶的，如果比顶的大，直接用顶就行，也就是所谓的单调栈

那么我们要准备两个，一个是从左往右的最大的，一个是从右往左的最大的

```python 
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        #准备两个，一个从后往前，一个从前往后
        left_max = [0]*n
        right_max = [0]*n
        left_max[0] = height[0]
        right_max[n-1] = height[n-1]
        for i in range(1,n):
            left_max[i] = max(left_max[i-1],height[i])
        for i in range(n-2,-1,-1):
            right_max[i] = max(right_max[i+1],height[i])
        ans = 0

        #主遍历
        for i in range(1,n-1):
            ans+= max(0,min(left_max[i-1],right_max[i+1])-height[i])
        return ans 


```
