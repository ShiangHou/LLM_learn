## 前缀和


假如我们有  2 3 5 6 1 8 9 ，k是8

我们构建前缀和，即从0位置到当前位置的总和

即前缀和是

2 3  5  6 1  8   9

2 5 10 16 17 25 34

我们的目标是找到i到j的和为8的，i到j上的和就是    Pj -P （i-1）

比如我想找 5 6的和，5位置是i，6位置是j，那么其实是 16-5，就是后面一个的位置减去i-1的位置

那么我们其实要找的就是 Pj -P （i-1） = k的就行

转换一下视角，当我们遍历到j的时候， 需要看有多少是P （i-1）+ k = P（i）的就行

但是，这样做的话，其实依旧是需要遍历前面所有的，位置，时间复杂度依旧是On2啊


## hash优化

换句话说，我们遍历到i的时候，只需要找到，前面的位置有哪几个的前缀和是等于Pj-k的就行

我们直接用hash，遍历一个前缀和存起来，因为每一次遍历的时候hash都是唯一的，遍历到i，直接从hash里面查就行


```python 
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        cache = {0:1}#优化时间的，同时用来存放前缀和的,需要初始化0为1
        n = len(nums)
        temp_sum = 0#辅助构建前缀和的
        ans = 0
        for i in range(n):
            #首先计算当前的前缀和是多少
            temp_sum+=nums[i]
            #然后去前缀和里面找有没有等于的前缀和-k的,有的话加上答案
            if temp_sum-k in cache:
                ans += cache[temp_sum-k]
            #无论有没有，都要记录一下当前的前缀和到cache里面吧
            if temp_sum in cache:
                cache[temp_sum] += 1
            else:
                cache[temp_sum] = 1
        return ans
```
注意，需要初始化hash的0为1 ，从而让前缀和为k的正好能加上去
