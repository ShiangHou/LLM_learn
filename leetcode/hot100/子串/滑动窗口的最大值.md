首先还是先说一下题目


提问给了一个数组，然后一个k值，这个k值就是滑动窗口的长度

然后，这个长为k的滑动窗口不断在数组里面滑动，因为每次都滑动一个嘛，

在这个期间，我们需要记录一下每次滑动时，这个k个窗口中的最大值是多少，每次都统计下来，然后存在ans中，最后输出

这道题的关键在于怎么去做这个窗口的最大值的维护，答案是用一个单调队列

1,3,-1,-3,5,3,6,7

什么意思，比如说1 3 -1

准备一个队列，然后开始放入1

3来了，发现比1大，然后让1滚蛋，

从而，最后队列里面的是 3 -1，注意到，我们的队列0位置始终是这个滑动窗口的最大值，什么意思，在1 3 -1的时候，3最大，那么3之前的哪些小角色都不重要，因为我滑动的时候，即使把3前面的滑动走了，那么不影响此时我的窗口里面最大的3

那为什么-1要留着，因为3后面的我需要保证是单调减的，因为如果3划走了，那么-1就要顶上去当老大哥

模拟一遍

然后开始滑动，1走，-3进来

1:需要看一下走的nums[i]是不是队列的最大值，不是的话就不用管

2：进来一个，依旧是按照我们之前的规则来去看，-3是小于-1的，直接放在队尾就行

此时这个窗口的最大值依旧是q[0]，每滑动一次，记录一下q【0】行就行


下一步，3出，5进

1：首先是3出，发现3走了，此时我们需要popleft，即从队列的左侧（就是正常队列，从右边append，左边出）把3弄出去

2：然后5进入，依旧是，把队列的比5小的全部弄出去，，最后只留下五

此时5在q【0】位置 ，继续输出就行


这样，5前面的 -1 和-3都是小角色，遍历到他们的时候无关紧要

```python 
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        n = len(nums)
        from collections import deque

        #先准备一个双端队列
        q = deque()

        ans = []

        #首先先把前k个先弄进去，
        for i in range(k):
            while q and nums[i] > q[-1]:#就是说，如果q不是空的，而且新加的比队列尾巴的要大，就让里面的滚
                q.pop()
            #队列为空，或者没有人家大的话，老实进入
            q.append(nums[i])
        l,r = 0,k-1#用来滑动的
        #先把第一个窗口的答案给加入
        ans.append(q[0])
        while r <n-1:
            #这里注意一下，是从当前窗口开始，眺望下一个的，所以是加入了r+1,斩杀掉l，这样的滑动方式,从而r应该约束到n-2位置，这样加入到n-1的时候正好结束
            #加入r+1的位置
            while q and nums[r+1] >q[-1]:
                q.pop()
            q.append(nums[r+1])

            #然后看弹出左边的是不是最大值
            if nums[l] == q[0]:#是的话就左边斩杀掉
                q.popleft()
            #不是的话不用管了，弹出的都是小垃圾，本来也没有记录
            #最后记录当前窗口的最大值，以及滑动
            ans.append(q[0])
            l+=1
            r+=1
        return ans
```
