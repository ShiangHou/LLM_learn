## 初步思路

这道题说，讲数组中是字母异位词的分组，那么这个的思路其实要复杂一些，首先我需要在字符与字符之间相比较才可以，

比如说，我拿到第一个字母，构建一个hash，然后遍历下一个，如果下一个是和我当前字符同的，那么就把他俩都加入到我的列表中，然后继续遍历，

在这个过程中，我可以去存一个数组，表示0和1，1表示该字符已经和别人配好对了，0表示还没配好

先写一个judge(s1,s2):是不是移位的

然后思想是这样，我先遍历一个，存到temp里面，然后从i往后开始遍历，把和自己一样的全部搜集，并且used打上，最后遍历完搜集到ans里面，开始遍历下一个，然后遍历到下一个的时候看一下used有没有被用过

```python 
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        n = len(strs)
        if n == 1:
            return [strs]
        used = [0]*n
        ans = []
        temp = []
        std = strs[0]#先拿0开始作比较
        for i in range(n):
            if used[i]== 1:
                continue
            else:
                #用当前的作比较，往后全部扫一遍
                std = strs[i]
                temp = [strs[i]]
                used[i] = 1#记录一下当前的
                for j in range(i+1,n):
                    #比较两个是不是
                    if self.judge(std,strs[j]):#如果是,就存起来
                        temp.append(strs[j])
                        #记录一下用过了
                        used[j] = 1
                    #如果不是，继续比较就行
                #往后扫完一遍后，temp已经存下来了自己和自己同组的，存入ans中
                ans.append(temp)
        return ans


    def judge(self,s1,s2):#判断s1和s2是不是异位词
        if len(s1) != len(s2):
            return False
        judge = {}
        for s in s1:#先把第一个字符放进hash统计一下
            if s in judge:
                judge[s] += 1
            else:
                judge[s] = 1
        for s in s2:#开始遍历第二个
            if s not in judge:#如果发现没有这个，直接报错
                return False
            judge[s] -= 1
            if judge[s] <0:#如果减完发现是负的，也报错
                return False
        return True#所有的保证 s2都在里面出现过，并且数量减去对应的没有出现负数，就是对的
```

分析一下时间复杂度，最坏的情况是，每次遍历，都需要和后面所有的匹配计算一下，

每次匹配，是遍历了两个s，假设字符串长度是m，那么每次匹配是Om，然后总体是On方的复杂度，那么最后就是On方*m的复杂度


## 更优答案

比如说 ate 或者eat，遍历的时候先把他们排序，排序完成后都是aet，然后遍历到的，如果说遇到一样的，直接合并到最后的答案里面就像

首先我们建立一个字典是collections.defaultdict()

然后遍历一个

```python 
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        mp = collections.defaultdict(list)
        for s in strs:
            key = "".join(sorted(s))#排完序拼接，这个是关键
            mp[key].append(s)
        return list(mp.values())#最后输出是values()这个也注意一下
```
