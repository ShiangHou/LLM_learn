# 两数之和

## 初步思路

梦开始的地方，

题目说，给了一个数组，给了一共target，然后需要找到数组中和为这个target的，然后返回数组下标就行

思路是，首先我准备一个dict，key是数组的值，value是下标，

然后我遍历一遍数组，如果数字没有出现，那么就加一下，如果目标i数字以及它的target-i也在dict里面，直接返回下标和dict里面存的下标

```python 
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        a = {}
        for i in range(len(nums)):
            if target-nums[i] in a:
                return [a[target-nums[i]],i]
            if nums[i] not in a:
                a[nums[i]] = i
```

注意，这里需要先判断target-nums[i]在不在里面，然后再看要不要存
