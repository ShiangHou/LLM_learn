# 找到字符串中所有字母异位词

力扣 438

## 初步思路

这道题说，有一个字符串s，是长的字符串，一个字符串p，是短的字符串，要求找到在长字符串中的，和算字符串所有的字母异位词，然后只需要返回开始的下标就行了

例如abab ab，那么需要返回 0 1 2 ，即从0开始的后两位，1开始的，2开始的

这题首先想到的是暴力解，首先需要去遍历每一个位置， 然后构建一个长度和len（p）相同的长度，然后进一个出一个，如果不在的话，直接划走，如果在的话，直接返回就行

我们可以给那个p构建出一个set，每一个元素，然后滑动的个时候进一个出一个就行

嘶，用set还是hash，如果p有重复的话，还是不能用啊

此外也不能简单的判断遍历的s中在不在里面，如果一直在，比如同为是 ab，然后遍历到一个aa，也是一直在，那么自然也就不行

所以还是要用hash，遍历到，然后元素-1，如果发现元素已经是0了还遍历到了，自然也是不行

为了方便，每次都重新搞一个hash，但是要一直用，用完就丢掉

## 代码优化版本

每次用原来的dict减去太麻烦了，我直接新构建一个dict，如果这个dict和p的一样，那不就是说明已经成功了嘛哈哈哈


```python 
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        if len(s)<len(p):#如果还没p长，直接斩杀
            return []
        ans = []
        #先给p构建一个dict，不用set是因为防止p出现多个
        p_dict = {}
        for i in p:
            if i not in p_dict:
                p_dict[i] = 1
            else:
                p_dict[i] +=1
            
        n = len(s)
        n_p = len(p)
        s_dict = {}
        #开始遍历，
        for right in range(n):#遍历右边的
            if s[right] in s_dict:#把右边的新的加入s的dict
                s_dict[s[right]] += 1 
            else:
                s_dict[s[right]] = 1 
            #看看左边应该在哪，在开始的时候，左边可能还没有，还在负数，
            left = right-n_p+1
            if left < 0:#第一个长度还没到
                continue
            else:#如果到了，就判断一下
                if s_dict == p_dict:#发现是一个答案
                    ans.append(left)
                #不是答案的话，也不用管，正常去把左边的在字典中去掉就行
                s_dict[s[left]] -=1
                #注意，如果减到0了，直接删除掉这个就行
                if s_dict[s[left]] == 0:
                    del s_dict[s[left]]
        return ans

```
