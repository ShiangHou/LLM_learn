# 无重复字符的最长字串

力扣3

## 初步思路

这题问的是说，有一个字符串，然后需要找到字符串中最长的，没有重复的子串，

这里的子串不是子序列，所以一个简单的想法就是说，用两个指针维护一个窗口，fast指针不断地往前探路，如果新来的和里面的出线了重复的，那么直接缩短slow，直到没有重复的

那么这里的问题在于说，我每次新来一个，都要去判断有没有和之间的有重复，怎么去判断这个有没有重复的行为，如果每次都去判断的话，时间复杂度就会炸掉，

假设，abcb，我们需要做两件事，首先是判断第二个b进来后有没有组成，其次是我的slow需要减少到哪才行，

首先第一个，判断有没有重复，这个肯定是不能直接全遍历一遍，可以额外用一个hash来去存，这样找的时间复杂度就是O1，

其次是减少，因为要求是连续的，所以新来的b后，slow一定要减少到之前的b

所以整体思路就是，slow和fast从0开始，fast往前推，加入set，并判断，如果set里面有，那么直接让slow往前缩，直到缩到 出现了了这个有的，最后每出现了这种情况一次，就记录ans max


```python 
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        cache = set()
        slow = 0
        ans = 0
        for i in range(len(s)):#这个就是fast不断往前推
            if s[i] not in cache:#如果这个是不重复的，直接加，然后继续
                cache.add(s[i])
                ans = max(ans,len(cache))#这里统计的全都是不重复的
                continue
            else:#出现重复的了，需要缩slow
                while s[slow]!= s[i]:#当slow位置上不是重复的，就一直往前走，走一步去掉一个set
                    cache.remove(s[slow])
                    slow +=1
                #跳出循环时，是走到相等的位置上，还需要再走一步，并且去掉尾巴的slow，以及加上i位置的相同的,
                slow+=1#其实不需要加，因为本来i和slow位置的就是相同的，
        return ans
```
