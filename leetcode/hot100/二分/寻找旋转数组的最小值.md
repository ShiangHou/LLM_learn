# 寻找旋转排序数组的最小值
要求logn，所以先遍历一遍肯定不行

直接就是二分，不管怎么说

哎，这个最小值，如果在中间的话，那么一定是右边的比自己大，左边的也比自己大

如果发现右边的比自己小，左边的比自己大

## 思路

这里是直接和rihgt比较就行了

如果mid比right小的话，说明右边就不用考虑了，并且右边是递增的

如果mid比right大的话说明在右边出现了折叠，那么直接在右边找


```python 
class Solution:
    def findMin(self, nums: List[int]) -> int:
        #如果当前比r的小，说明右边单调递曾，答案在左边
        #如果当前比r大，说明右边是折叠的，答案在右边
        l,r = 0,len(nums)-1
        while l <= r:
            mid = (l+r) // 2
            if nums[mid]<nums[r]:
                r = mid#r自己也有可能是
            else:
                l = mid +1

        #最后不断逼近的r就是的
        return nums[r]

```
