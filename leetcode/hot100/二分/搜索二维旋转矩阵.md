# 搜索二维旋转矩阵

## 初步思路

首先，本质上，如果我们可以把这个矩阵压成一个数组就行，

对于一个1维的，转换就行，

假设长度是m*n-1

那么转换后，就是我们需要到看看在矩阵的行和列是多少


首先看一下行，index，就是index//n

然后看一下列，列就是index%n

```python 
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        #将这个矩阵拍扁成一维的就行，注意转化一下一维和二维的关系

        m,n = len(matrix),len(matrix[0])
        l,r = 0,m*n-1
        while l <=r :
            mid = (l+r)//2
            #需要讲这个一维的转化为二维的矩阵坐标
            row = mid // n
            col  = mid %n
            if matrix[row][col] == target:
                return True
            elif matrix[row][col] > target:
                r = mid-1
            else:
                l = mid+ 1
        return False


```
