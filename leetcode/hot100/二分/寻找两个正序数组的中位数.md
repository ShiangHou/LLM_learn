# 寻找两个正序数组的中位数

但这道题难就难在：我们二分查找的对象，不是具体的数字，而是一刀「切口」（Partition）。

降维打击：寻找完美的“一刀”想象一下，把这两个有序数组拼在一起，中位数的作用是什么？

中位数的作用，就是把一个集合划分为长度相等的左右两半，并且保证左半边的最大值永远小于等于右半边的最小值。

既然两个数组本身就是有序的，我们只需要在 nums1 里切一刀（位置记为 i），在 nums2 里切一刀（位置记为 j），把它们各自切成左右两半。

为了满足中位数的定义，这两刀必须满足两个死条件：长度守恒：nums1 左半边元素个数 + nums2 左半边元素个数 = 总长度的一半。这意味着，只要我们在 nums1 中确定了切口 i，nums2 中的切口 j 是自动推导出来的公式：j = (m + n + 1) // 2 - i。

交叉小于等于：nums1 切口左边的值（暂称 L1）必须 $\le$ nums2 切口右边的值（R2）。

nums2 切口左边的值（暂称 L2）必须 $\le$ nums1 切口右边的值（R1）。

只要我们通过二分查找，在较短的那个数组里找到了满足上述条件的切口 i，这道题就解出来了！

厚米的意思是，我需要在这两个数组上找到两个位置，i和j，使得i小于j就行

比如 1 5 7 8 9

2 3 4 6 10

我的i是在第一个的5位置，j是在第一个的6位置，这样就行了

然后在较短的上面找到符合条件的就行
```python 
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        #在短的上面找到i，
        if len(nums1) > len(nums2):
            nums1,nums2 = nums2,nums1
        m,n = len(nums1),len(nums2)
        #数组的一半是 （m+n+1）//2
        half_len = (m+n+1) // 2
        l,r = 0,m
        while l <= r:
            i = (l+r) // 2
            j = half_len-i
            #找到那四个临界点
            l1 = nums1[i - 1] if i > 0 else float("-inf")
            r1 = nums1[i] if i < m else float('inf')
            l2 = nums2[j-1] if j > 0 else float('-inf')
            r2 = nums2[j] if j < n else float('inf')
            #后面的其实就是约束了一下不在数组范围的
            if l1 <= r2 and l2 <= r1:
                #找到啦，然后看一下奇偶
                if (m+n)%2 == 1:#奇数
                    #取i和j最大的
                    return max(l1,l2)
                else:
                    #左边部分最大和右边部分最小
                    return (max(l1,l2)+min(r1,r2))/2
            elif l1 > r2:
            # L1 太大了，说明 nums1 的切口太靠右了，必须向左收缩
                r = i - 1
            else:
                l = i+1




```
