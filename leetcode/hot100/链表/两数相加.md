# 两数相加

## 初步思路，弄一个进位就行
从开始的val相加，如果小于10，直接接上新的链表
如果大于10，弄一个进位，往后传递，如果最后还是有进位，直接加在链表后面
直接对10取%就行，

```python 
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        head = None#最后返回的头节点
        cur = None #不断往后传递的
        carry = 0#  需要进位
        while l1 or l2:
            val1 = l1.val if l1 else 0#如果是空，直接弄成0
            val2 = l2.val if l2 else 0
            temp_sum= val1+val2+carry
            #接在后面的是直接和，然后取余，
            cur_val = temp_sum%10
            #需要carry的是和除10
            carry = temp_sum // 10

            #处理一下开头
            if head == None:
                head = ListNode(cur_val)
                cur = head
            else:
                cur.next = ListNode(cur_val)
                cur = cur.next
            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None
        if carry != 0:#最后如果说还是有
            cur.next = ListNode(carry)
        return head




```
