# 随机链表的复制

## 初步思路

这道题说，让我们去复制一个链表，但是有难点

这里的链表结构是，除了next，还有一个叫做randon的，他是随机指向链表中任意一个节点

所以说，我们采用交织-解耦的方法

首先我们先构建一个链表 A -A‘-B-B’

然后，A‘的next当然是指向B

那么A’的random呢，假设A的random指向C，那么，C是指向C‘的

所以A’的random指向A.random.next就行！！！！

然后我们把链表拆开，奇就是原链表，偶就是新链表


```python 
class Solution:
    def copyRandomList(self, head: 'Optional[Node]') -> 'Optional[Node]':
        if not head :
            return None
        #首先建立A-A’
        cur = head
        while cur:
            _next = cur.next
            cur.next = Node(cur.val)
            cur.next.next = _next
            cur = cur.next.next
        #然后把所有的‘的random都指向源节点的randon
        cur = head
        while cur:
            _next = cur.next.next#这个指的是下一个真节点
            copy = cur.next#这个就是假的节点
            copy.random = cur.random.next if cur.random else None#假的节点的random是真的random的下一个
            cur = _next
        
        #然后再解耦
        cur = head
        ans = head.next
        while cur.next:
            _next = cur.next.next#这个指的是下一个真节点
            copy = cur.next
            cur.next = _next
            copy.next = _next.next if _next else None
        return ans
```
