# 排序链表

## 初步思路

这题其实就是用归并排序去排链表

既然是归并排序，那其实和之前一样，搞一个递归就行

首先我们需要找到节点的中间，用一个快慢指针找到，

然后slow是左边最后一个，我们需要找到右边第一个，即slow.next，然后记得把左右切开

递归调用左，递归调用右

然后merge左右

merge是什么！就是我们的“合并两个有序链表啊！”哈哈哈哈


```python 
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        slow,fast = head,head.next
        #在回文链表里面，我们其实是从head出发的，但是当时的slow是第二段的起点，现在我们要左边的最后，所以要从fast为head.next开始才可以
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        #此时slow是左边的最后一个，
        right = slow.next
        slow.next = None#切开
        l = self.sortList(head)
        r = self.sortList(right)
        return self.merge(l,r)
    def merge(self,node1,node2):#合并两个有序链表，并返回最初的头
        if node1 and not node2:
            return node1
        if not node1 and node2:
            return node2
        if not node1 and not node2:
            return None
        head = node1 if node1.val <= node2.val else node2#选好小的当头
        #两个链表的起始位置
        cur = head 
        cur1 = head.next#两个链表的起始位置
        cur2 = node2 if head == node1 else node1
        while cur1 and cur2:
            if cur1.val <= cur2.val:
                cur.next = cur1
                cur1 = cur1.next
            else:
                cur.next = cur2
                cur2 = cur2.next
            cur = cur.next
        cur.next = cur1 if cur1 else cur2
        return head
```
