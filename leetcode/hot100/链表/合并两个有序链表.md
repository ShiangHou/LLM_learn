# 合并两个有序链表
力扣21

## 初步思路

合并两个有序链表，我们需要准备几个变量，一个是cur1，一个是cur2，用来在两个链表上进行对比，这两个变量是只在自己的链表上走

然后准备一个cur不断的在新生成的链上走

然后准备一个head最后接住

首先是谁小谁当head，然后小的cur1 = cur.next
然后cur移到当前上（cur1或者cur2）

最后记得，如果cur1或者cur2为None了话，直接把另一个加到尾巴上

```python 
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        if list1 == None and list2 != None:
            return list2
        if list1 != None and list2 == None:
            return list1
        if list1 == None and list2 == None:
            return None 

        head = list2 if list1.val >= list2.val else list1
        cur = head
        cur1 = head.next
        cur2 = list2 if head == list1 else list1
        #注意这里起始位置的设立，cur1设立为head的下一个，cur2设立为另一个
        while cur1 and cur2:
            if cur1.val >= cur2.val:
                cur.next = cur2
                cur2 = cur2.next
            else:
                cur.next = cur1
                cur1 = cur1.next
            cur = cur.next
        #最后弹出的时候有一个是None了
        cur.next = cur1 if cur1 else cur2
        return head




```
