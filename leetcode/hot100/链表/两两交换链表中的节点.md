# 两两交换链表

就是把相邻的两个元素交换

这个需要一点小巧思，

我们直接进入第二种写法，依旧是准备两个变量，pre和cur，

我们的目标是将cur和cur.next进行反转

所以说，cur.next应该是指向cur的，然后cur是指向原来的cur.next.next,pre指向cur.next，然后整体往后移动（pre指向cur，然后cur往后移动

A    B      C
pre cur cur.next

现在要
A    C     B
          pre


首先需要存一下pre

```python 
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if head == None or head.next == None:
            return head
        pre = head
        cur = head.next
        _next = cur.next
        #然后现在需要交换 pre和cur
        cur.next = pre#让cur的next指向pre
        pre.next = self.swapPairs(_next)#然后pre指向下下一个，递归，已经转换好的
        return cur#最后返回尾巴

```
